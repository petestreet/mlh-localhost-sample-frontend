About this app

This app demonstrates a frontend and backend working together in order to query data from Twitter's API and display it in a new format.

The frontend is a static site that uses the Axios library to make AJAX requests, and the VueJS framework to display data reactively. Find out how to run it locally on the Git repository.

The backend of this app is written with Node.js and Express. Find out how to run it locally on its Git repository.
